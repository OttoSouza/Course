DOM é Document Object Model, é o que o browser usa para mostrar a pagina da web ou appp e em  javascript é facil de manipular a DOM.
Demora muito tempo para que as alterações do Dom aconteçam. O navegador precisa fazer duas operações realmente intensivas. Uma é repintar. 
Isso é alterar um elemento e adicionado a uma página e depois exibir novamente, que é para recalcular o layout da página e mova as coisas, se necessário.
Então, mudar o Dom foi uma operação muito cara.
